# Fonts Directory

## Custom Font Setup

Place your `dirty-english.ttf` font file in this directory for the custom font to load properly on the website.

### Instructions:
1. Upload the `dirty-english.ttf` font file to this directory
2. Ensure the file name is exactly `dirty-english.ttf` (lowercase)
3. Deploy your changes to your hosting provider for the font to take effect on the live site
